# GPT-2 Next Word Prediction

This project fine-tunes a **GPT-2 language model** using the Hugging Face Transformers library to perform **next word prediction** on a text dataset.  
The trained model learns contextual patterns in language and predicts the most probable next word in a sequence.

Such models are useful for applications like:

- Autocomplete systems
- Writing assistants
- Text generation
- Language modeling

---

## Project Overview

This project implements a full NLP pipeline:

1. Text preprocessing and tokenization
2. Dataset preparation for transformer training
3. Fine-tuning GPT-2 using Hugging Face Trainer API
4. Evaluation using perplexity and prediction accuracy
5. Generating next-word predictions from trained model

---

## Model

Model Architecture: **GPT-2 (Transformer based language model)**  
Framework: **Hugging Face Transformers**  
Deep Learning Backend: **PyTorch**

---

## Training Details

Tokenizer: `AutoTokenizer`  
Model: `GPT2LMHeadModel`  
Training Framework: Hugging Face `Trainer`

Training dataset size: **45,000+ text sequences**

Training configuration:

- Epochs: **2**
- Task: **Next Word Prediction**
- Hardware: **Google Colab CPU/GPU**

---

## Performance

Final Model Metrics:

Top-3 Prediction Accuracy: **~70%**  
Perplexity: **~25**

The model learns contextual language patterns and generates coherent next-word predictions.
